Doctests¶

If you add the doctest extension to your conf.py (add sphinx.ext.doctest to the extensions list), you can run tests on your documentation to guarantee that it is correct.

You will likely also need to modify the sys.path within conf.py so that your module is on the path – after all, it needs to be loaded when it’s being tested.
